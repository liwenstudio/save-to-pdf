# 🐱 Save Whole Webpage to PDF  
A Chrome extension that saves the current webpage as an **A4 multi‑page PDF**, with layout scaled to match what you see on screen.

Designed with an original hand‑drawn cat icon by **Liwen** 🐱✨

---

## 🌟 Features
- Save any webpage as **A4 multi‑page PDF**
- Layout scales based on **current screen proportion**
- Avoids awkward page breaks
- Works on most websites (Wix, blogs, portfolios, articles)
- Simple one‑click UI

## 📦 Installation (Load Unpacked)
1. Download this repository as ZIP  
2. Unzip it  
3. Open Chrome → `chrome://extensions/`  
4. Enable **Developer mode**  
5. Click **Load unpacked**  
6. Select the folder

## 🧾 How It Works
The extension injects a print‑optimized stylesheet into the current page:

- Forces A4 page size  
- Scales layout to match screen proportion  
- Avoids breaking images or sections  
- Triggers Chrome’s native print dialog
